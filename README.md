# FinFlow Frontend - Expense Tracker

## Overview
Modern, responsive expense tracking dashboard built with vanilla JavaScript and Chart.js.

## Features
- ✅ User authentication (login/register)
- ✅ Expense tracking with categories
- ✅ Recurring expenses
- ✅ Bill reminders with calendar
- ✅ Split expenses with friends
- ✅ Financial analytics with charts
- ✅ Multiple currency support
- ✅ Light/Dark theme
